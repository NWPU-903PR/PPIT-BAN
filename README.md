# PPIT-BAN

This repository comprises the source code, datasets, and models associated with our research paper entitled: 
Predicting protein-protein interaction with interpretable bilinear attention network.

# Dependencies
* python = 3.8
* pytorch
* numpy
* pandas
* dgl
* sklearn
